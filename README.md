# 실행 방법

테스트 할 폴더에 진입 후

깃 클론 실행

`git clone https://github.com/taelee42/Prime_factorization_Tester.git`

file structure:

|---실행 프로그램 폴더

 			|--------Tester



클론한 폴더에 진입후

`./PrimeFactoTester.sh`



# 기능

- 작은 숫자 (2~20)까지 실행
- unsigned int max value(4294967295) 테스트
- unsigned int 바깥 값 테스트
- 인자가 2개이상일 때 테스트
- 숫자가 아닌 문자가 들어왔을 때 테스트
- 음수 테스트
